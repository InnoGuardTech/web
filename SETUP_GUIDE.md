# دليل التثبيت والاستخدام - حراج اليمن الفاخر v4.1

## 🚀 متطلبات التثبيت

- **PHP**: 7.4 أو أعلى
- **MySQL**: 5.7+ أو MariaDB 10.3+
- **Apache**: مع mod_rewrite و mod_headers
- **HTTPS**: مُوصى به (مطلوب للـ PWA)

---

## 📋 خطوات التثبيت

### 1. استنساخ المستودع

```bash
git clone https://github.com/InnoGuardTech/web.git
cd web
```

### 2. إعداد البيئة

انسخ ملف `.env.example` إلى `.env` وقم بتحديث المتغيرات.

### 3. تثبيت قاعدة البيانات

```bash
php scripts/db_setup_enhanced.php
```

### 4. ضبط الصلاحيات

```bash
chmod -R 755 uploads/
chmod 644 .env
chmod 644 .htaccess
```

### 5. تشغيل التطبيق

```
http://localhost/frontend/index.php
```
